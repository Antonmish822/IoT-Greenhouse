# Полный функционал умной теплицы на основе ESP32

---

## **Базовый (критически необходимый) функционал:**

**1. Климат-контроль (ядро системы):**

*   **Температура:**
*   
    *   Мониторинг: датчики внутри (3-4 точки) и снаружи.
    *   Управление: вентиляторы, вентиляционные заслонки с сервоприводами (охлаждение), трубный/инфракрасный обогреватель (нагрев).
*   **Влажность воздуха:**
    *   Мониторинг: гигрометры внутри.
    *   Управление: вытяжной вентилятор (снижение), ультразвуковой увлажнитель (повышение).
*   **Освещение:**
    *   Мониторинг: фотодатчик (освещенность в люксах).
    *   Управление: реле для фитоламп (досветка по таймеру/датчику).

**2. Управление поливом:**
*   **Мониторинг:** датчики влажности почвы на ключевых грядках.
*   **Управление:** электромагнитные клапаны на магистралях полива. Полив по расписанию или при падении влажности ниже порога.

**3. Безопасность и оповещения:**
*   Аварийные уведомления (Telegram/email) при выходе параметров за критические пределы.
*   Датчик протечки воды под стеллажами.

---

## **Расширенный (дополнительный) функционал:**

**4. Продвинутый климат-контроль:**
*   **Солнечная радиация:** пиранометр для управления затеняющими сетками/шторами (автоматическое затенение при пиковой нагрузке).
*   **Температура листа/почвы:** инфракрасный датчик для точного определения стресса растений.
*   **CO₂:** контроль уровня углекислого газа, управление подачей CO₂ из баллона для ускорения фотосинтеза.
*   **Дифференцированный микроклимат:** независимое управление зонами с разными культурами.
*   **Автопроветривание по алгоритму:** комбинирование данных температуры, влажности и точки росы для избегания конденсата.

**5. Питание растений и гидропоника:**
*   **Фертигация:** управление насосом-дозатором для внесения жидких удобрений в поливную воду по расписанию.
*   **Гидропоника:** контроль EC (электропроводность) и pH питательного раствора с автоматической коррекцией.

**6. Автоматизация и прогнозирование:**
*   **Сумма активных температур (САТ):** автоматический расчет для прогноза сроков созревания.
*   **Интеграция с прогнозом погоды:** использование API для упреждающих действий (закрыть теплицу перед дождем, включить обогрев перед заморозком).
*   **Солнечный коллектор:** управление насосом для прокачки теплоносителя через темный бак-аккумулятор.

**7. Мониторинг и аналитика:**
*   **Логгирование:** запись всех данных на SD-карту или в облако с построением графиков.
*   **Учет ресурсов:** мониторинг расхода воды и электроэнергии.
*   **Дистанционный мониторинг:** веб-интерфейс и мобильное приложение для просмотра данных и ручного управления.

**8. Защита растений:**
*   **УФ-стерилизация:** ночное включение УФ-ламп для подавления патогенов в воздухе.
*   **Система опрыскивания:** управление помпой и форсунками для автоматического распыления биопрепаратов.
*   **Камера с ИК-подсветкой:** визуальный мониторинг роста растений и обнаружение вредителей.

**9. Механика и обслуживание:**
*   **Контроль состояния оборудования:** датчики тока на двигателях форточек и насосах для предупреждения о заклинивании.
*   **Автоматическая мойка:** управление системой очистки крыши и стен от пыли (насосы, распылители).
*   **Роботизированная платформа:** базовый интерфейс для управления мобильным роботом для обработки или мониторинга.
