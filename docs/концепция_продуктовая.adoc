= Концепция продукта

Ключевая идея: Один координатор на всё СНТ (садоводство) разворачивает mesh-сеть, к которой по принципу "подключи и работай" присоединяются дешевые конечные устройства соседей.

== Как это работает

1. Инфраструктура (общая для поселка)

- Один базовый модуль с LTE-модемом и eSIM ставится у председателя или самого продвинутого дачника
- Устройства общаются между собой по mesh-сети (каждая новая точка усиливает сигнал)
- Интернет и облачный сервис оплачиваются коллективно (или "скидываются" соседями)

2. Устройства для каждого (индивидуальные)

- Конечные модули (управление клапанами, датчики, реле) — покупаются каждым дачником под свои задачи
- Одноразовые картриджи (датчики влажности почвы) — меняются каждый сезон за копейки
- Все устройства автоматически находят mesh-сеть и регистрируются в системе

3. Управление (простое и понятное)

- В приложении или личном кабинете пользователь выбирает готовые сценарии из библиотеки:
"Защита от заморозков", "Полив по расписанию", "Курятник", "Погреб"
- Настраивает только параметры: температура +5°C, время полива 8:00, влажность 70%
- Больше ничего не нужно программировать

4. Коллективный эффект

- Первый дачник ставит координатор — остальным подключение почти бесплатно
- Чем больше соседей в сети, тем надежнее mesh-покрытие
- Можно делиться данными (например, одна метеостанция на всех)

== Главные преимущества

- ✅ Дешево для каждого — один модем на всех  
- ✅ Просто для пенсионеров — готовые сценарии, не надо программировать  
- ✅ Надежно — mesh-сеть работает, даже если у кого-то выключили свет  
- ✅ Модульно — докупай только то, что нужно (клапан, картридж, датчик)  
- ✅ Социально — соседи помогают друг другу покрытием и поддержкой  

Итог: Умный огород перестает быть "игрушкой для гиков" и становится такой же обычной вещью, как общий столб с электричеством или водопровод на всю улицу.
