

=== Регистрация и вход
Пользователь создаёт аккаунт или входит в существующий.

*Функции:*

* Регистрация по email и паролю
* Вход в систему
* (Опционально) вход через Yandex/ VK / Google / Apple

*Элементы интерфейса:*

* Поля: Email, Пароль
* Кнопки: «Войти», «Зарегистрироваться», «Забыли пароль?»

=== Панель управления теплицами (Dashboard)
Главная страница после входа. Отображает все привязанные теплицы.

*Информация по каждой теплице:*

* Название (например, «Теплица на даче»)
* Статус подключения: ✅ Онлайн / ⚠️ Оффлайн
* Текущая температура (°C)
* Текущая влажность (%)
* Состояние привода: 🪟 Открыт / 🚪 Закрыт / 🔄 В движении

*Действия:*

* Кнопка «Добавить новую теплицу»
* Клик по карточке → переход на детальный просмотр

=== Детальный просмотр теплицы
Страница управления одной теплицей с поддержкой линейного привода.

*Основные блоки:*

==== Текущие показатели
* Крупные цифры: температура, влажность
* Индикатор состояния привода:

  * **Положение**: 0% (закрыто) — 100% (открыто)
  * **Статус**: «Закрыто», «Открыто», «Открывается...», «Закрывается...»

==== Графики
* График температуры за последние 24 часа
* График влажности за последние 24 часа
* (Опционально) график положения привода

==== Управление приводом

Кнопки:

- [ Полностью открыть ]
- [ Полностью закрыть ]
- [ Остановить ] (активна только во время движения)

Переключатель режима:

- [ ] Ручной режим
- [x] Автоматический режим

Настройки автоматики (в автоматическом режиме):

* «Открывать при температуре выше ___ °C»
* «Закрывать при температуре ниже ___ °C»
* Ползунки или числовые поля (диапазон: 10–40°C)
* (Опционально) гистерезис: «Минимальная разница между порогами — 2°C»

=== Уведомления и события
Отдельная вкладка или панель с историей событий.

*Примеры событий:*

* «22 мая, 14:30 — Температура 31°C. Начато открывание привода.»
* «22 мая, 14:32 — Привод достиг положения 100%.»
* «22 мая, 18:05 — Температура упала до 24°C. Начато закрывание.»
* «21 мая, 22:00 — Привод не отвечает (ошибка питания/механики)»

*Настройки уведомлений:*

- Включить/выключить оповещения
- Выбрать каналы:

  - [x] В приложении
  - [x] Email
  - [ ] Telegram
  - [ ] SMS

=== Добавление нового устройства
Процесс привязки физического устройства к аккаунту.

*Способы добавления:*

* Ввод **Device ID** и **Access Token** вручную
* Сканирование **QR-кода** с наклейки на устройстве

*Подтверждение:*

* После добавления — автоматический переход на страницу новой теплицы
* Первые данные (температура, положение привода) появляются в течение 30 секунд
